# NgCovid19

This is a simple project for getting the data on _Covid-19_ pandamic. I used angular for this app

## Development server

To start devolping:

|     | NPM           | Yarn           |
| :-- | :------------ | :------------- |
| 01. | `npm install` | `yarn install` |
| 02. | `npm start`   | `yarn start`   |

The dev server will start at `http://localhost:4200`
